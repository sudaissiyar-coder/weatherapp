# SkyCast - Weather App

A beautiful, responsive weather application that provides real-time weather information using the OpenWeatherMap API.

## 🌟 Features

- **City Search**: Search for weather by city name
- **Geolocation**: Get weather for your current location
- **Real-time Data**: Live weather updates with temperature, humidity, wind speed, pressure, and more
- **Responsive Design**: Works on desktop and mobile devices
- **Weather Icons**: Visual weather icons from OpenWeatherMap

## 🚀 How to Use

1. **Open the App**: Open `index.html` in your web browser
2. **Get API Key**: 
   - Sign up at [OpenWeatherMap](https://openweathermap.org/api)
   - Get your free API key
   - In `app.js`, replace `YOUR_OPENWEATHER_KEY` with your actual API key
3. **Search Weather**: 
   - Type a city name and click search
   - Or click "📍 My Location" for current location weather

## 🛠️ Technologies Used

- **HTML5**: Structure and layout
- **CSS3**: Styling and responsive design
- **JavaScript (ES6+)**: Functionality and API integration
- **OpenWeatherMap API**: Weather data source

## 📱 Demo

You can view a live demo by opening `index.html` in any modern web browser.

## 📝 Notes

- Make sure to replace the placeholder API key in `app.js` with your own OpenWeatherMap API key
- New API keys may take 1-2 hours to activate
- The app requires an internet connection to fetch weather data
